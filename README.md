# VIM Configure

### Install build version

https://gitee.com/pegasuswiki/vimrc/tree/master

### Manual Install

1. Install dependencies:

    - git
    - make


2. Run the following commands as normal user:

    ```
    git clone https://gitee.com/pegasuswiki/vimrc.git
    cd vimrc
    make
    ```


### Uninstall

    ```
    git clone https://gitee.com/pegasuswiki/vimrc.git
    cd vimrc
    make clean
    ```
